# QRVoice

Δωρεάν εφαρμογή Android που σκανάρει QR με κείμενο και το εκφωνεί αυτόματα στα ελληνικά, χρησιμοποιώντας την υπηρεσία Text-to-Speech του κινητού.

## Μορφή QR
Το QR μπορεί να περιέχει απλό κείμενο:

    Καλώς ήρθατε στο μουσείο.

ή προαιρετικά:

    SPEAK:Καλώς ήρθατε στο μουσείο.

## Δημιουργία APK με Android Studio
1. Εγκαταστήστε το Android Studio.
2. Ανοίξτε τον φάκελο `QRVoice`.
3. Περιμένετε να ολοκληρωθεί το Gradle Sync.
4. Επιλέξτε **Build > Build APK(s)**.
5. Το APK θα δημιουργηθεί στο `app/build/outputs/apk/debug/app-debug.apk`.

## Δημιουργία APK με GitHub Actions
1. Δημιουργήστε νέο δημόσιο ή ιδιωτικό repository στο GitHub.
2. Ανεβάστε όλα τα αρχεία του φακέλου.
3. Ανοίξτε **Actions > Build APK > Run workflow**.
4. Μετά την ολοκλήρωση, κατεβάστε το artifact `QRVoice-debug-apk`.

## Εγκατάσταση στο κινητό
1. Μεταφέρετε το `app-debug.apk` στο κινητό.
2. Ανοίξτε το από την εφαρμογή **Αρχεία**.
3. Επιτρέψτε προσωρινά την εγκατάσταση από αυτή την πηγή, αν ζητηθεί.
4. Πατήστε **Εγκατάσταση**.
5. Ανοίξτε το QRVoice και επιτρέψτε την κάμερα.

## Ελληνική φωνή
Στο Android ανοίξτε:
**Ρυθμίσεις > Πρόσθετες ρυθμίσεις / Σύστημα > Γλώσσες και εισαγωγή > Έξοδος μετατροπής κειμένου σε ομιλία**
και εγκαταστήστε ελληνικά δεδομένα φωνής, εάν λείπουν.
